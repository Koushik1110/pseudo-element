Pseudo element

Web developing is a work of art. all the small things in the web developing contributes to the structure of a website.A web developer needs to use his sense to add, remove, restructure, ease of use and a lot of things to make a website.So every single things we learn in CSS, no matter how big or small it is ha a big contribution in the website. One of those element is called"pseudo element".

Hover

The first topic of this lecture is called "pseudo element".There are a lot of pseudo elements in css. but the first thing we will learn about is hover.Hover is a function that allows the text to change its form factor when mouse cursor is placed on the things that is customized with hover function.We can also do a lot of other things such as change the font size, add new color and a lot other things.
